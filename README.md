# QMSPilot Corrective Action CAPA Northstar

Production-oriented QMSPilot corrective action and CAPA application designed for the Northstar Digital Toolbox.

## Core capabilities
- CAPA intake, containment, investigation, root cause analysis, corrective action, preventive action, effectiveness verification, COPQ, evidence, approvals, and closure
- Northstar workforce routing for Pilot, Atlas, Forge, Sentinel, and Vector
- Northstar host bridge, direct API submission, and local offline queue
- Draft save/load, JSON import/export, PDF printing, email summary, validation, and audit trail
- Responsive branded interface for desktop, tablet, and mobile

## Architecture direction
This repository is the beginning of the shift from standalone HTML microtools to Northstar production applications. Shared authentication, tenant context, UI components, workforce routing, evidence handling, audit logging, and API contracts will be standardized for reuse across future QMSPilot applications.
